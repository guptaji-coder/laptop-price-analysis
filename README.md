# Laptop Price Analysis

## Objective
The objective of this project is to analyze laptop prices based on different features such as brand, RAM, storage, processor, and operating system.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
The dataset contains information about laptop specifications and their prices.

## Steps Performed
- Imported and cleaned the dataset
- Handled missing values
- Performed exploratory data analysis (EDA)
- Visualized price trends based on different features

## Results
The analysis shows how different hardware specifications impact laptop pricing.

## Conclusion
This project helps in understanding market trends and factors affecting laptop prices.
